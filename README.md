# Required Tools

```bash
# install GoLang
https://go.dev/doc/install

# install revive linter 
go install github.com/mgechev/revive@latest
```

# Getting Started (MacOS)

```bash
# Running Tests (All Challenges)
go test ./...

# Running Linter (All Folders)
revive ./...
```
